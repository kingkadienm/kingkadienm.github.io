# 知识体系

# 移动端开发

Android

  • 基础：自定义View、AIDL、知识点收集
  • 进阶：MVVM架构、协程、Room数据库、性能优化
  • 实战：完整项目架构、最佳实践

Flutter

  • 基础：Dart语法、Flutter配置
  • 进阶：状态管理、自定义Widget、动画
  • 实战：完整应用开发、项目结构

# 前端开发

Vue.js

  • 基础：实例、模板、组件、生命周期
  • 进阶：路由、状态管理、性能优化
  • 实战：项目结构、最佳实践、完整示例

React

  • 基础：JSX、组件、State、Props
  • Hooks：useState、useEffect、自定义Hooks
  • 进阶：状态管理、路由、性能优化
  • 实战：项目架构、测试策略

JavaScript

  • 基础：变量、函数、对象、异步编程
  • 进阶：闭包、this绑定、设计模式、性能优化

CSS

  • 基础：选择器、布局、定位、动画
  • 进阶：预处理器、架构模式、性能优化

# 后端开发

Python

  • 基础：语法、数据结构、面向对象
  • 进阶：装饰器、异步编程、Web开发
  • 实战：Flask/Django项目、测试、部署

Go

  • 基础：语法、并发、接口
  • 进阶：反射、上下文、并发模式
  • 实战：RESTful API、项目架构、测试

# 学习路径

每个技术栈都遵循以下学习路径：

  2. 进阶篇 - 深入理解高级特性和最佳实践
  3. 实战篇 - 通过完整项目巩固知识
